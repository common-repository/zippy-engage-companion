A plugin to add the Zippy Engage embed code to your site.

== Description ==

The Zippy Engage plugin is a tool for adding our Zippy Courses Embed code to your WordPress site.

The Zippy Engage Plugin requires an account on ZippyEngage.com in order to be used. Make sure you set up and authorize your site in ZippyEngage.com alongside installing this plugin.

Simply install the plugin, and this plugin will connect to your Zippy Engage account and verify that your current site is registered. If it is, the Zippy Engage embed code will be added to every front-facing page in your site. You can confirm that your site is connected properly by logging into your WordPress admin panel and selecing Tools --> Zippy Engage.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/zippy-engage-companion directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Under "Tools" → "Zippy Engage", you can confirm that your site has been connected properly to Zippy Engage. If it hasn't, follow the instructions on this page to get started.

== Frequently asked questions ==

= Do I have to have a Zippy Engage account to use this? =

Yes, an account on https://zippyengage.com is required in order to use this plugin. Otherwise, our embed script will reject any requests.

= I have a Zippy Engage account, but I'm still seeing a warning that my site hasn't been connected. =

After you create an account, make sure that you've also added your Site to Zippy Engage. Visit "Tools" → "Zippy Engage" in your wordpress admin for instructions on getting set up.
